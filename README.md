# SuperStore-Sales-Insights-Dashboard

# Project Overview
The SuperStore Sales Insights Dashboard is an interactive Power BI project designed to analyze sales performance across regions, product categories, and time periods.  
It helps stakeholders understand trends, identify top-performing segments, and make data-driven business decisions.

# Objectives
- Analyze overall sales and profit performance  
- Track regional and category-wise sales trends  
- Identify top and low-performing products  
- Support strategic decision-making using visual insights  

# Tools & Technologies
- Power BI – Dashboard development & visualization  
- Power Query – Data cleaning and transformation  
- DAX – Measures and calculated KPIs  
- Data Source – SuperStore sales dataset  

# Key Features
- Total Sales, Profit, and Quantity KPIs  
- Region-wise and State-wise sales analysis  
- Category and Sub-category performance comparison  
- Time-based trends (monthly / yearly analysis)  
- Interactive filters and slicers for dynamic exploration
